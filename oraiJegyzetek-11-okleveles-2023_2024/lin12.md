Scripting

- python script: `#!/usr/bin/env python3 app.py`
- bash script: `#!/bin/bash script.sh`

Hashbang: `#!` - a script futtatásához használt interpreter elérési útja

A=1 --> jó
A = 1 --> rossz
ALMA=1 --> jó
echo $ALMA --> 1
echo $A$ALMA --> 11
echo $A1$ALMA --> 111

PosX: Egy szabvány, hogy egy os miért lesz használható.
